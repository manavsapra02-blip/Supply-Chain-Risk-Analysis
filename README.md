# 📦 Supply Chain Risk & Optimization System

## 🎯 Project Overview
This project analyzes logistical performance using a *synthetic dataset of 100,000 records*. The objective was to identify shipping bottlenecks and visualize delivery risks to improve global supply chain efficiency.

## 🚀 Key Insights
- *On-Time Delivery:* Identified a critical performance rate of *37.8%*.
- *Regional Risk:* The *North Region* was identified as the highest-risk area for logistical delays.
- *Latency Analysis:* Calculated an average shipping delay of *3.49 days* across all modes.
